Coding

On this page you will learn how to code using html.

Starting to code

Go to brackets.io, Download and open Brackets

Make 2 files #1 named index.html #2 named style.css To make stlye.css you have to go to the bottom left of brackets and right click and then press new file and then name it style .css

Then on index.html write <!DOCTYPE html> and on the outside write your arrows.

After that write <html> now, after you typed that something really cool happend, the tag doubled! But there is 1 little difference between the opening and closing of the tag, the slash. The slash represents the closing tag. But after every tag that will eventually have another tag in it you make, you have to press enter.

Now make a head tag (<head>) then inside the head tag you write <title>Here you write whatever you want your website name to be (not the domain).</title>

then press the lightning bottonon the top right for a live preview of your website.
